GitHub 자동 동기화

  • GitHub 저장소 기반 자동 동기화 추가
  • 단원·폴더·정렬·이름 변경·퀴즈 및 서술형 학습 기록·임시 답안 동기화
  • 단원 추가와 학습 기록 변경 시 GitHub 자동 업로드
  • 앱 실행 시 및 1분 간격 GitHub 최신 데이터 자동 확인
  • 업데이트 HTML을 GitHub에 등록하고 다른 기기에서 자동 감지·적용
  • 기기별 GitHub 토큰 로컬 저장 및 백업·원격 데이터에서 토큰 제외
  • 수동 올리기·내려받기·연결 테스트와 동기화 상태 표시
  • 기존 단원·학습 기록·데이터 스키마 1.0 유지 및 롤백 지원
최초 연결 안내
업데이트 적용 후 앱의 GitHub 연결 메뉴에서 저장소와 fine-grained token을 한 번 설정해야 합니다. 이미 사용 중인 다른 기기에도 최초 한 번 1.5.0을 직접 적용해야 이후 업데이트가 자동 동기화됩니다.
