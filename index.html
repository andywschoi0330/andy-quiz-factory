Andy의 퀴즈 공장 업데이트 1.4.0

appId: andy-quiz-factory스키마 1.01.3.0 이상 호환

기존 퀴즈 단원은 자동으로 quiz 유형으로 유지됩니다. 서술형 기록과 임시 답안은 별도 메타데이터로 저장되며 기존 단원·학습 기록·폴더·정렬 정보는 변경하지 않습니다. 적용 전 현재 번들이 보관되어 롤백할 수 있습니다.

변경 내역

  • 서술형(essay) 단원 유형과 기존 퀴즈(quiz) 자동 호환 추가
  • 메인 단원 카드에 부드러운 파란색 퀴즈 배지와 초록색 서술형 배지 추가
  • 쉬움·보통·어려움 단계별 타이핑 빈칸 학습 및 개별·전체 정답 확인 추가
  • 전체 답안 작성, 핵심어 가중치 기반 참고 점수, 포함·누락 핵심어 및 답안 비교 추가
  • 줄별 암기, 틀린 줄 재학습, 전체 답안 연결 보기 추가
  • 답안 구조 순서 학습과 모바일용 위·아래 이동 버튼 추가
  • 핵심어 플래시카드와 모름·애매함·암기함 자기 평가 추가
  • 첫 글자·초성·일부 공개·정답 공개의 5단계 힌트 추가
  • 핵심어·문장 뒷부분·한 문장·전체 답안 가리기 학습 추가
  • 말하기식 자기 점검과 기억 정도 기록 추가
  • 서술형 학습 기록·임시 답안·복습 필요 항목 저장 및 백업·복원 지원
  • 통합 랜덤과 오답 복습에 전체·퀴즈·서술형 유형 필터 추가
  • 기존 단원·폴더·정렬·이름 변경·학습 기록·오답 기록 보존 및 롤백 지원
